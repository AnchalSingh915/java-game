🎮 Java Game Repository: Fun Coding Projects 🚀

Welcome to the Java Game Repository! 🕹️ This collection of small Java games will help you practice and sharpen your Java skills while having fun! These games combine logic, user interaction, and creativity to create a learning environment that is both engaging and educational.

📝 Description

This repository includes three fun Java games:

Synonym Finder Game:
A simple game where the user has to guess the correct synonym of God Krishna’s names. The program checks the user's input and gives feedback accordingly. 🕵️‍♂️✨
Snake Game Simulation:
A dice-based snake game where the player moves on a board by selecting a dice roll between 0 and 5. The program tracks the player’s position, and if they land on certain positions, they face a penalty! 🐍🎲
Pattern Printing Game:
A fun text-based pattern printing game that takes a string as input and prints different patterns based on whether the string's length is even or odd. 🎨🔠
🎮 What's Inside?
Synonym Finder Game
The user is prompted to enter synonyms of Lord Krishna’s names, and the game checks whether the input matches any of the predefined names.
Snake Game Simulation
A simple dice-based game where the player's position is updated based on the dice value, with certain positions causing a "snake" penalty.
Pattern Printing Game
This program reads a string input and prints a dynamic pattern based on whether the string's length is even or odd.
